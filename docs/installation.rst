============
Installation
============


Stable release
--------------

To install Kane Text Editor, run this command in your terminal:

.. code-block:: console

    $ pip install kane_text_editor

This is the preferred method to install Kane Text Editor, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Kane Text Editor can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/tobyontour/kane_text_editor

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/tobyontour/kane_text_editor/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/tobyontour/kane_text_editor
.. _tarball: https://github.com/tobyontour/kane_text_editor/tarball/master
